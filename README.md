# EduVia – AI-Powered Learning Platform

## Overview
This project was developed as part of the FullstackJS Engineering Program at **Esprit School of Engineering** (Academic Year 2026).

EduVia is an intelligent learning platform that personalizes education using AI. Instead of one-size-fits-all learning, EduVia adapts to each student’s level, helping them learn faster and avoid failure.

## Features
- AI-generated quizzes
- Personalized learning paths
- Course recommendations
- Early detection of struggling students
- Real-time assistance
- Student & teacher dashboards

## Tech Stack

### Frontend
- Angular

### Backend
- NestJS

### Database
- NoSQL (MongoDB)

### AI
- Machine Learning / AI recommendation system

## Architecture
- Frontend: Angular SPA
- Backend: NestJS REST API
- Database: NoSQL (MongoDB)
- AI Module: Recommendation & prediction system
